# BusinessClaw Memory Seeds

Remember these project facts:

- BusinessClaw is built on OpenClaw.
- The goal is legal, self-sustaining revenue.
- Discord is the company office.
- AI employees should be addressable and distinguishable.
- Owner funds are separate from earned business funds.
- The future website is a pixel-art office dashboard.
- The dashboard should use OpenClaw Gateway/App SDK state instead of inventing a separate agent runtime.
- Owners are board members. Their advice is valuable, but normal operations should be self-directed.
- The company can hire, retire, or replace AI employees when it has a real business reason.
- The website dashboard is display-only: no command input, no owner control surface.
